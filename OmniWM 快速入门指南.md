# OmniWM 快速入门指南

在本教程中，你将在 macOS 上安装并配置 OmniWM，体验自动窗口平铺、快捷键导航、工作区管理等强大功能，让你的 macOS 桌面焕然一新。

<Note>
本教程大约需要 15-20 分钟完成。你不需要任何平铺式窗口管理器的经验，我们会从零开始。
</Note>

## 你将学到什么

完成本教程后，你将能够：

- 在 macOS 上成功安装 OmniWM
- 使用快捷键在窗口之间快速导航
- 切换不同的布局模式（Niri 和 Dwindle）
- 使用 Quake 终端快速访问命令行
- 配置应用程序规则自定义窗口行为

## 准备工作

在开始之前，确保你满足以下条件：

- 一台 Mac 电脑，运行 macOS 15.0 (Sequoia) 或更高版本
- 管理员权限（用于安装应用程序）
- 基本的 macOS 使用经验

<Warning>
OmniWM 需要授予辅助功能权限才能管理窗口。如果你之前没有授予过任何应用此权限，系统会在首次启动时提示你。
</Warning>

## 第一步：安装 OmniWM

打开终端，使用 Homebrew 安装 OmniWM。如果你还没有安装 Homebrew，可以从 https://brew.sh 获取安装方法。

```bash
brew tap BarutSRB/tap
brew install omniwm
```

<Note>
如果你无法使用 Homebrew，可以从 GitHub Releases 页面下载 `OmniWM.zip`，解压后将 `OmniWM.app` 移动到 `/Applications` 文件夹。
</Note>

安装完成后，你应该在应用程序文件夹中看到 OmniWM。

## 第二步：启动 OmniWM 并授予权限

在 Finder 中导航到 `/Applications` 文件夹，找到 `OmniWM.app` 并双击启动。

首次启动时，系统会弹出一个对话框，提示你需要授予辅助功能权限。点击 **打开系统设置**。

<Steps>
  <Step title="打开辅助功能设置">
    在系统设置中，点击 **隐私与安全性** > **辅助功能**。
  </Step>
  <Step title="找到 OmniWM">
    在辅助功能页面中，找到 **OmniWM** 的开关。
  </Step>
  <Step title="启用权限">
    将 OmniWM 的开关设置为开启状态。系统可能需要你输入管理员密码确认。
  </Step>
</Steps>

<Warning>
如果你的 Mac 启用了 SIP（系统完整性保护），OmniWM 仍然可以正常工作，不需要关闭 SIP。

授予权限后，OmniWM 会自动开始管理你的窗口。你应该看到所有窗口自动排列成列布局。
</Warning>

## 第三步：体验窗口自动平铺

现在打开几个应用程序，比如 Safari、Terminal 和 Notes。你应该立即看到：

- 窗口自动排列成垂直列（Niri 布局）
- 每个应用程序占用一部分屏幕空间
- 无需手动调整窗口大小

如果你没有看到自动平铺，点击菜单栏上的 **O** 图标，确保 **Enable tiling** 已开启。

## 第四步：使用快捷键导航

OmniWM 的强大之处在于完全使用键盘管理窗口。以下是几个最常用的快捷键：

### 在窗口之间移动

按住 **Option** 键，然后按 **方向键** 在窗口之间移动焦点：

- `Option + ←` - 聚焦到左边的窗口
- `Option + →` - 聚焦到右边的窗口
- `Option + ↑` - 聚焦到上方的窗口
- `Option + ↓` - 聚焦到下方的窗口

<Note>
OmniWM 使用 Option 键作为修饰键，这与 macOS 的标准快捷键不冲突。
</Tip>

### 移动窗口位置

按住 **Option + Shift** 然后按方向键，可以将当前窗口移动到指定位置：

- `Option + Shift + ←` - 将窗口移到左边
- `Option + Shift + →` - 将窗口移到右边

### 切换布局模式

按 `Option + Shift + L` 可以在两种布局模式之间切换：

- **Niri（滚动列）**：窗口垂直排列成多列，每列可以包含多个堆叠窗口或标签页
- **Dwindle（二叉树分割）**：经典的平铺布局，新窗口将空间二分

尝试切换几次，感受两种布局的不同之处。

## 第五步：使用 Quake 终端

Quake 终端是一个从屏幕边缘滑下的下拉式终端，非常适合需要频繁使用命令行的用户。

按 **Option + `**（反引号，位于 ESC 键下方）可以快速调出 Quake 终端。

<Note>
OmniWM 的 Quake 终端基于 Ghostty 构建，提供流畅的动画和优秀的性能。
</Note>

在 Quake 终端中，你可以：

- 使用 `Cmd + T` 打开新标签页
- 使用 `Cmd + D` 水平分割窗格
- 使用 `Cmd + Option + 方向键` 在窗格之间导航

再次按 **Option + `** 可以隐藏 Quake 终端。

## 第六步：管理工作区

工作区帮助你将不同的应用程序分组到不同的虚拟桌面。

### 切换工作区

- `Option + 1-9` - 切换到工作区 1-9
- `Option + Shift + 1-9` - 将当前窗口移动到指定工作区
- `Option + Ctrl + Tab` - 在当前和上一个工作区之间切换

### 打开工作区栏

按 `Option + Shift + O` 可以打开概览模式，查看所有工作区和一个窗口的缩略图。

## 第七步：探索更多功能

OmniWM 还有许多其他实用功能：

### 窗口查找器

按 `Ctrl + Option + Space` 可以快速搜索并跳转到任何窗口。输入应用程序名称或窗口标题进行模糊搜索。

### 随时呼出菜单

- `Ctrl + Option + M` - 在光标位置显示当前应用的原生菜单
- `Ctrl + Option + Shift + M` - 显示可搜索的菜单面板

### 隐藏状态栏图标

右键点击菜单栏的 OmniWM 图标，可以隐藏或显示状态栏图标。
