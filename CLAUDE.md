# CLAUDE.md

本文件为 Claude Code (claude.ai/code) 在本仓库中工作时提供指导。

## 仓库概述

这是一个 **Obsidian 知识库** - 基于 Markdown 的个人知识管理系统。笔记按以下目录组织：

- **我的项目** - 项目文档 (antagent, anteam, memory, plato, 全自动营销产品)
- **学习笔记和教程** - 学习笔记 (Python, Rust, LangChain, Claude Code, ast-grep 等)
- **技术工作** - 技术文档 (Redis, StarRocks, Java GC, 微服务架构等)
- **操作记录** - 安装配置指南 (Neovim, tmux, WezTerm, Karabiner 等)
- **曾用过的提示词** - 可复用的提示词模板
- **datatist** - 工作相关文档

## 知识库组织规范

- 笔记主要使用中文
- 优先使用 Mermaid 做图表
- 复杂图表使用 Excalidraw（存储为 `.excalidraw.md` 文件）
- 项目文档通过 wiki-links 相互引用

## Git 工作流

本知识库使用 Git 备份。提交信息应使用简洁的中文描述变更内容。
