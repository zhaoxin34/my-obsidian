## 🚀 快速开始

### 安装环境
```bash
# 安装LangChain
pip install langchain

# 安装LangGraph
pip install langgraph

# 完整安装
pip install langchain[all]
```

### 运行示例
```bash
# 进入示例目录
cd examples/

# 运行基础示例
python basic_workflow_example.py

# 运行条件路由示例
python conditional_workflow_example.py

# 运行多Agent示例
python multi_agent_workflow_example.py
```
