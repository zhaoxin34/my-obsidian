```mermaid
flowchart TB
    subgraph DATA["📦 数据层"]
        USER["

        "]

        **内容系统**
        • 文案生成

    end

    %% 数据流向
    USER --> DECISION
    USER --> PROFILE
    USER --> ANALYSIS
    EXTERNAL --> ANALYSIS
    KB --> DECISION

    %% 系统间流向
    DECISION --> REACH
    REACH --> ANALYSIS
    ANALYSIS --> FEEDBACK
    FEEDBACK --> DECISION

    PREDICT --> DECISION
    PREDICT --> PROFILE
    PROFILE --> DECISION
    CONTENT --> REACH

    %% 样式
    classDef data fill:#e1f5fe,stroke:#01579b
    classDef core fill:#fff3e0,stroke:#e65100
    classDef support fill:#f3e5f5,stroke:#4a148c
    classDef feedback fill:#e8f5e9,stroke:#1b5e20

    class USER,EXTERNAL,KB data
    class DECISION,REACH,ANALYSIS core
    class PREDICT,PROFILE,CONTENT support
    class FEEDBACK,LOOP feedback
```

## 反馈闭环详解

```mermaid
flowchart TB
    START["📌 决策制定"] --> STEP1["制定营销策略"]
    STEP1 --> STEP2["选择目标人群"]
    STEP2 --> STEP3["确定触达时机"]
    STEP3 --> STEP4["准备触达内容"]

    STEP4 --> EXEC["🚀 执行触达"]
    EXEC --> MONITOR["📊 效果监控"]
    MONITOR --> ANALYZE["分析数据"]

    ANALYZE --> ASSESS{"效果评估"}
    ASSESS -->|达标| SUCCESS["✅ 达成目标"]
    ASSESS -->|未达标| ISSUE["⚠️ 问题诊断"]

    SUCCESS --> FEEDBACK["📝 总结经验"]
    ISSUE --> ROOT["🔍 根因分析"]
    ROOT --> ADJUST["🔧 策略调整"]

    FEEDBACK --> IMPROVE["💡 优化改进"]
    ADJUST --> IMPROVE
    IMPROVE --> START

    classDef start fill:#bbdefb
    classDef exec fill:#fff9c4
    classDef monitor fill:#e1f5fe
    classDef decision fill:#f3e5f5
    classDef improve fill:#e8f5e9

    class START,STEP1,STEP2,STEP3,STEP4 start
    class EXEC exec
    class MONITOR,ANALYZE monitor
    class ASSESS,SUCCESS,ISSUE,ROOT decision
    class FEEDBACK,ADJUST,IMPROVE improve
```
