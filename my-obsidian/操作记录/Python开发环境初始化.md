## 使用poetry 管理项目 vs code 插件安装

### 必需扩展

1. Python (Microsoft)

- 提供 Python 语言支持、调试、IntelliSense

- 扩展 ID: ms-python.python

1. Pylance (Microsoft)

- Python 语言服务器，提供类型检查、自动补全

- 通常随 Python 扩展一起安装

- 扩展 ID: ms-python.vscode-pylance

## 推荐扩展

1. Ruff (Astral Software)

- 快速 Python linter 和格式化工具（项目已配置）

- 扩展 ID: astral-sh.ruff-vscode

1. Black Formatter (Microsoft)

- Black 格式化支持（项目已配置）

- 扩展 ID: ms-python.black-formatter

1. Pytest (Microsoft)

- Pytest 测试支持

- 扩展 ID: ms-python.pytest

1. Python Test Explorer

- 可视化测试运行器

- 扩展 ID: littlefoxteam.vscode-python-test-adapter
